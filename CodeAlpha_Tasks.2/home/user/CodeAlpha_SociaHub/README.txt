# 🌐 SocialHub – Mini Social Media Platform

SocialHub is a full-stack social media web application built with Django. It allows users to create accounts, manage profiles, share posts with images, like posts, comment on posts, and follow other users. The application provides a clean, responsive interface with real-time interactions using JavaScript and AJAX.

⚙ Installation
------------------------------------------------------------

1. Clone the repository

git clone <repository-url>

2. Open the project

cd SocialHub

3. Create Virtual Environment

python -m venv venv

4. Activate Virtual Environment

Windows

venv\Scripts\activate

Linux / Mac

source venv/bin/activate

5. Install Dependencies

pip install -r requirements.txt

6. Apply Migrations

python manage.py migrate

7. Create Superuser

python manage.py createsuperuser

8. Run the Server

python manage.py runserver

Visit

http://127.0.0.1:8000/
